Detailed Description

Attributes can be attached to any object inheriting from class Attributed. Built-in attributes are provided by cfglib, and new ones can be defined by its users. Every attribute has a name and a type. An attribute should implement a method clone (attributes are duplicated when attached) and method Print for debugging purposes. Attributes can be serialisable (subclass SerialisableAttribute) or not (subclass NonSerialisableAttribute).
