24 
25  DAAInstruction
26 
27 This abstract class is a generic definition of an instruction when
28 used to compute the contents of registers for the determination of
29 addresses of load/stores.
30 
31 All the concrete instructions (or rather classes of instructions) are
32 derived from this abstract class.
33 
34 Instructions are grouped into different categories depending on
35 the way they modify registers (nop, load, store, add, move,
36 etc). There is a concrete instruction class corresponding to each
37 category.
38 
39 The correspondance between instruction names (mnemonics in asm code)
40 and instruction categories for address analysis (DAAInstruction *) is
41 done in InstructionType stored in Arch_dep (see files in
42 ArchitectureDependent directory).
43 
44 Rq: The implementation is done for MIPS architecture only !!!
45
