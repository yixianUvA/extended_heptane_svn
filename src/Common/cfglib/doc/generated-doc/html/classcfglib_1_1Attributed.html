◆ GetAttribute()

Attribute& cfglib::Attributed::GetAttribute ( std::string  symbol)

Get a attribute given its name. The method makes a copy of the attribute by calling its method "clone" before the attribute is stored. All attributed must have a 'clone' method implemented. Method HasAttribute has to be called before trying to retrieve an attribute to make sure the attribute with name 'symbol' exists (else, an assertion in the code fails). In the case the attribute contains pointers on other attributes, the attribute copy constructor has to be defined. Example of use of attributes (with n an attributed object) IntegerAttribute ia(6); // Declare an integer attribute n.SetAttribute("myinteger",ia); // Attach a copy of the attribute to object n

IntegerAttribute ia; // Declare an attribute assert(n.HasAttribute("myinteger");// Make sure n has an attribute attached ia=n.GetAttribute("myinteger"); // Retrieve a copy of the attribute in ia int val = ia.GetValue(); // Get it's value (here, a simple integer)
