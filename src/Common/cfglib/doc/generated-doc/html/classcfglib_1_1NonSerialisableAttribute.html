Detailed Description

NonSerialisableAttribute are local attributes that are not expected to be serialised in XML files. Ther are built-in NonSerialisableAttribute (integers, unisgned long, string, BigInteger). New non serialisable attributes can be defined by inheriting from class NonSerialisableAttribute
